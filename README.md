# STEP LAB. Jr. — 小学生向けAI学習プラットフォーム

小学校の **国語・算数・英語**（小1〜小6）を、AIが伴走しながら学べる**独立したWebサービス**です。
ClaudeアプリやClaudeアカウントは不要。**URLにアクセスするだけ**で、PC / Chromebook / iPad / スマートフォンのブラウザから利用できます。

学習指導要領（平成29年告示）の内容構成に基づいて単元を整理し、
**学ぶ → 練習 → 確認テスト → AI復習 → クリア** のステップで「誰一人取り残さない学び」を目指します。

---

## 🚀 クイックスタート（ローカル / 校内サーバー）

```bash
npm install
npm start
# → http://localhost:3000
```

Node.js 18以上が必要です（better-sqlite3 のビルドに build-essential / python3 が必要な環境があります）。

## ☁️ デプロイ（Render の例）

1. このフォルダをGitHubにpush
2. Render で「New → Web Service」→ リポジトリを選択
3. Build Command: `npm install` / Start Command: `npm start`
4. 環境変数を設定（下記）。**Persistent Disk を `/data` にマウント**し `DATA_DIR=/data` を設定するとDBが永続化されます
5. HTTPSはRenderが自動で終端します

Railway / Fly.io / さくらVPS + nginx などでも同様に動作します。

## 🔧 環境変数

| 変数 | 必須 | 説明 |
|---|---|---|
| `JWT_SECRET` | ◎ | ログイントークンの署名鍵。長いランダム文字列を設定（未設定だと再起動でログアウトされます） |
| `PORT` | − | 待受ポート（既定 3000） |
| `DATA_DIR` | − | SQLite保存先ディレクトリ（既定 `./storage`） |
| `ANTHROPIC_API_KEY` | − | 設定すると「こぶこぶ先生」がClaudeによる本格AIチャットになります。未設定でもルールベースの学習ガイドとして動作します |
| `AI_MODEL` | − | 使用モデル（既定 `claude-sonnet-4-6`） |

`.env` は使わずプラットフォームの環境変数機能で設定してください（`cp .env.example` を参考に）。

## ✅ 実装済み機能

- **クラスコード**: 教師が登録→「新しいクラスを作る」で4文字コードを発行。生徒は登録時または設定でコード入力するとクラスに参加。学年団での共同管理（同一コードに複数教員参加）も可能
- **宿題**: 教師が「学年→単元→期限→メモ」で出題。生徒のホームに📌表示され、単元テスト80点以上で✅完了。教師画面で「完了◯/◯人」を確認・削除可能
- **教員ダッシュボード**: クラスの生徒一覧（今日の問題数/学習時間・正答率・クリア数・連続日数・🟢今日学習）、生徒詳細（苦手単元・宿題状況・28日カレンダー・バッジ・達成率）

- **アカウント**: メール+パスワードの新規登録 / ログイン / ログアウト / パスワード変更 / メール変更 / プロフィール編集 / パスワード再設定（トークン方式）
- **セキュリティ**: bcryptによるハッシュ化、httpOnly Cookie + JWT、ロールベース権限（生徒/教員/保護者/管理者の下地）
- **データ同期**: 学習履歴・正答率・XP・バッジ・復習予定などをサーバーDBに保存。**どの端末からログインしても同期**
- **カリキュラム**: 国語・算数・英語 全85単元（小1〜6）。各単元に「まとめノート」付き
- **出題エンジン**: 算数は難易度連動の**自動生成**（無限に練習可能）、国語は学年別漢字配当表ベースの読みクイズ+文法・語彙プール、英語は語彙ペアから双方向出題。**毎回Fisher–Yatesでシャッフル**
- **AIレベル調整**: 単元ごとの正答率で 初級→標準→応用→発展 を自動昇降
- **忘却曲線の復習**: テスト合格（80点以上）で 翌日/3日/7日/14日/30日後 の復習を自動予約
- **やる気の仕掛け**: XP・レベル・9種のバッジ・連続学習ストリーク・応援メッセージ・パーフェクト演出
- **ホーム**: 今日/今週の学習時間、直近正答率、苦手単元、復習キュー、続きから、おすすめ、AIメッセージ
- **マイページ**: 学習カレンダー（60日ヒートマップ）、苦手/得意ランキング、バッジ図鑑、達成率、AIコメント
- **AIチャット**: 右下に常駐。`ANTHROPIC_API_KEY`設定時は児童の学年・苦手単元を踏まえたClaude応答（会話履歴もDB保存）
- **支援教育モード**: 音声読み上げ（Web Speech API・速度変更可）/ 文字サイズ / 背景色変更 / 選択肢を2択に削減 / 問題数削減 / 集中モード / ダークモード
- **PWA**: ホーム画面に追加可能。静的ファイルをオフラインキャッシュし、通信回復後に同期
- **教員・保護者向け**: `/api/admin/*` に児童サマリーAPI（role=teacher/parent/admin）。UIは今後拡張予定

## ⚠️ できないこと・注意点（正直な申告）

- **メール送信は未接続**です（SMTP情報が環境依存のため）。パスワード再設定は画面上に再設定リンクを表示する方式で動作します。nodemailer等の接続箇所は `server.js` の `/api/auth/forgot` にコメントで明示しています。
- **AIチャットのフル機能にはAnthropic APIキーが必要**です（有料）。未設定時はルールベース応答になります。
- **Googleログインは非搭載**です（Google Cloudでの各校のOAuthクライアント発行が必要なため、要件どおりメール+パスワード方式を採用）。
- **私はデプロイの実行そのものは代行できません**（お客様のRender/GitHubアカウントが必要です）。上記手順どおりに進めれば10分程度で公開できます。
- 問題形式は現在 **四択 / 入力（一問一答・穴埋め相当）** に対応。並び替え・資料/地図/年表/画像問題はデータ構造（`t`フィールド）として拡張可能な設計です。
- 総ルビ（全文ふりがな）は形態素解析辞書が必要なため未実装。読み上げ機能で代替しています。
- 管理者アカウントは、登録後にDBで `UPDATE users SET role='admin' WHERE email='...'` を実行して昇格してください。
